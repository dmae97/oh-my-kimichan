## Reddit / Hacker News (medium)

```
Show HN: oh-my-kimichan — Turn Kimi Code CLI into a worktree-based coding team

Kimi CLI (v1.39+) is powerful for single-agent coding, but software
engineering is inherently parallel: architects design while coders
implement, reviewers audit while QA verifies.

oh-my-kimichan is an open-source harness that wraps Kimi CLI and
spawns specialized agents in isolated Git worktrees:

- Architect -> writes plan.md
- Coder -> implements in isolated worktrees
- Reviewer -> adversarial code review
- QA -> enforces lint / typecheck / test / build gates

Key differentiators:
- MCP server auto-detection from .omk/config.toml
- Google DESIGN.md -> UI code generation
- Live HUD for real-time team monitoring
- Safety hooks (blocks rm -rf, sudo, forced push)
- Auto-fixes Kimi CLI's chronic "Unhandled exception" PTY bug
- Custom welcome banner (PNG/JPEG/GIF support)

Zero cloud signup. Zero Docker. Just Node.js + Git + tmux.

npm install -g oh-my-kimichan
omk init && omk doctor && omk chat

Would love feedback from anyone using Kimi CLI daily.

https://github.com/dmae97/oh-my-kimichan
```
